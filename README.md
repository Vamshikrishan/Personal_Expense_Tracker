# 💰 Personal Expense Tracker

A full-stack web application to manage and track your personal expenses efficiently. Monitor your spending, categorize transactions, and gain insights into your financial habits.

## 🎯 Features

- ✅ Track daily expenses with detailed information
- ✅ Categorize transactions (Food, Transportation, Entertainment, etc.)
- ✅ View expense summaries and analytics
- ✅ Responsive design for mobile and desktop
- ✅ Real-time data updates
- ✅ Secure data storage with MongoDB
- ✅ User-friendly interface

## 📦 Installation

### Prerequisites
- Node.js (v14 or higher)
- MongoDB
- npm or yarn

### Steps

1. **Clone the repository**
   ```bash
   git clone https://github.com/Vamshikrishan/Personal_Expense_Tracker.git
   cd Personal_Expense_Tracker

2. **Install backend dependencies**
   npm install

3. **Install frontend dependencies**
   cd client
   npm install
   cd ..

4. **Set up environment variables Create a .env file in the root directory:**
   MONGODB_URI=your_mongodb_connection_string
   PORT=5000

5. **Start the application**
   npm run dev
